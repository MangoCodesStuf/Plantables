# How Plantables Works

Plantables is powered by a Microcontroller known as the BBC Micro: bit V2, and uses only a set of
2 AA batteries to power it. It works by using metal two pins, which are electrical conductors which will
send electricity through the soil without harming the plant and the microcontroller will messure the amount of
electricity that it recives and use that to calculate humidity. Soil is a great Isolator which means its great at not
letting electricity, but when we begin to add water electricity starts to flow. Too much water will give us a electricity 
value arround of 1203 volts. And no water will give us arround 0 volts, It might not be exactly 0 but a low number compared 
to higer Humidity levels. We can use these numbers to calculate and show humidity finding a balance between dry and over watered.
